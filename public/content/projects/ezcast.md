**基於 Multicast 的區網聊天系統**  
系統中包含一 Server 端裝置與多 Client 端裝置，  
系統啟動後會透過 multicast 取得 Server 端 IP 等資訊，並透過 Server 端做訊息轉送。

Server: [https://github.com/x3388638/EzCast_web_server](https://github.com/x3388638/EzCast_web_server)
